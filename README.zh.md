[FEC 兽展日历](https://www.furryeventchina.com) 是一个收集并展示中国大陆与兽迷相关的展览活动的网站。这是它的源代码仓库。

## 开始开发

```
yarn install
yarn dev
```
使用浏览器打开 http://localhost:3000 查看结果。
